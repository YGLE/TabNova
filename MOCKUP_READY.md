## ✨ Features du Prototype

### **Design System**
✅ Palette couleurs exacte (8 couleurs groupe)
✅ Typography (Inter 18px, 14px, 12px)
✅ Dark mode only (#000 background)
✅ Badges rouges (style Gmail)
✅ Spacing cohérent (8px base)

### **Animations**
✅ Particles background (slow moving stars)
✅ Bubble hover (scale 1.05 + glow)
✅ Item hover (border color change)
✅ Sync indicator (pulse animation)
✅ Smooth transitions (200ms)

### **Interactivité**
✅ Clic sur groupe → Alert
✅ Clic sur "Créer groupe" → Alert
✅ Clic "Ouvrir" → Alert
✅ Focus input → Blue border + glow
✅ Hover bubbles → Visual feedback

### **Responsive**
✅ Desktop: 2 colonnes côte à côte
✅ Tablet: Flexible layout
✅ Mobile: 1 colonne (vertical stack)
